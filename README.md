# BBS Mod

BBS Mod is a Minecraft mod for Fabric 1.20.4 and 1.20.1 (also compatible with Forge) that provides cinematic animation tools within Minecraft. Its primary purpose is to allow creators to build animated content such as videos, machinima, or in-game storytelling. For more information, visit the official [Modrinth](https://modrinth.com/mod/bbs-mod/) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/bbs-mod) pages.
